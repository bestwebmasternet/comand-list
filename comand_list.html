//Comand List//

Linux (ubuntu):

speedtest - первірка швидкості інтернету
wethr - погода
neofetch - інфо
cd~/Desktop - не прямий перехід до каталогу
cd .. - вийти в попередній каталог (батьківській каталог)
pwd - переглянути шлях до каталога
mkdir nazvakataloga- створення новго каталога
cd nazvakataloga - прямий перехід до каталога
echo "My text" > file.txt - запис тексту в файл та створення файлу
cat file.txt - прочитати вміст файлу (виведе в термінал)
rm file.txt - видалити файл
ls -la - відображає скриті каталоги та файли

PowerShell (windows):

ls -force - відображає скриті каталоги та файли

Git Comand:

git --version - перевірка версії git
git config --list - переглянути налоштування
git config --global user.name "Sasha Zimin" - зміна імені автора
git config --global user.email "info@zimin.dev" - вказуємо пошту автора
git init - створення нового репозиторія
git config --global init.defaulBranch 'name' - зміна головного імені гілки (master, main)
git status - переглянути файли, які знаходяться в індексі (файли що мали зміни, але ще не закомічені з робочої дерикторії)
git add. - додавання всіх змін в індекс (додавання змін)
git commit - збереження з індексу в репозиторій (збереження змін)
git checkout - перейти до певної версії

Git Status:

untracked - неможливо відстежити
unmodified - немодифікований
modified - модивікований
staged - підготовлений

Git Object:

blob - для файлів, файли, зберігає різні версії
tree - каталог, який містить інші каталоги та файли, а також файли в цих каталогах
commit - після збереження змін, містить в собі ім`я автора, пошту
annotated tag - створеннятегу за допомогою якого можна пересуватися поміж версій репозиторія
